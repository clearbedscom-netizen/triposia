# Content Generation Guidelines

## Prohibited Practices

### Writing Style
- **No generic explanatory paragraphs** - Avoid boilerplate explanations
- **No conversational AI tone** - Use technical, factual language
- **No marketing adjectives** - Avoid words like "best", "cheapest", "great option"
- **No filler phrases** - Remove phrases like "in conclusion", "overall", "as you can see"
- **No repetitive sentence structures** - Vary syntax across pages

## Content Generation Requirements

### Model Specifications
- **ONLY Meta LLaMA models** - No OpenAI GPT models
- **DATA-SUMMARIZATION MODE** - Model must operate in data-summarization mode only
- **Factual fields only** - Never invent text beyond existing factual fields
- **No competitor content** - No paraphrasing of competitor content
- **No hallucinated advice** - No guarantees or invented recommendations

## Implementation

These guidelines apply to all pages where content generation is required.
